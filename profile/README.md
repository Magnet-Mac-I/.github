## Overview

<a href="#">Magnet</a> adds window snapping and tiling to Mac, filling the gap left by macOS's limited native window management. The application enables the kind of precise window arrangement that Windows users take for granted — dragging to screen edges to fill halves, corners to fill quarters, and other positions — without requiring manual resizing by dragging window edges.

The <a href="#">drag-to-snap system</a> activates when you drag a window to the edge or corner of the screen. Approach the left edge and the window snaps to the left half. Drag to a corner and it fills that quarter of the screen. <a href="#">Top and bottom snapping</a> fills the upper or lower half of the display. These snap zones trigger with a brief pause at the screen edge, distinguishing intentional snapping from passing the cursor near the edge.

<a href="#">Keyboard shortcuts</a> perform the same tiling operations without any mouse interaction. Assign key combinations to each snap position — left half, right half, upper left quarter, and every other arrangement. Trigger window positioning from anywhere without touching the window title bar. <a href="#">Custom shortcut assignment</a> lets you define the key combinations that feel natural for your keyboard layout and existing shortcut habits.

The <a href="#">thirds layout</a> divides the screen into three vertical columns, useful on wide displays where halves produce windows that are still very wide. Position windows at left third, center third, right third, or left two-thirds and right two-thirds for flexible arrangements. <a href="#">Sixths positions</a> combine thirds horizontally with top and bottom halves for six tiling positions on large displays, enabling highly organized multi-window layouts.

## Key Features

- <a href="#">Edge drag snapping</a> positions windows to halves quarters thirds and other layouts by dragging to screen edges
- <a href="#">Corner snap zones</a> fill quarter-screen positions when windows are dragged to display corners
- <a href="#">Keyboard shortcut tiling</a> arranges windows to any position without touching the window or mouse
- <a href="#">Custom shortcut mapping</a> assigns your preferred key combinations to every available snap position
- <a href="#">Thirds layout support</a> divides displays into three columns for organized wide-screen arrangements
- <a href="#">Sixths position grid</a> combines thirds with top and bottom halves for six tiling zones on large screens
- <a href="#">Multi-monitor snapping</a> works across all connected displays with cross-screen window movement shortcuts
- <a href="#">Menu bar window control</a> provides access to all positions through a dropdown for non-keyboard use
- <a href="#">App exclusion list</a> prevents snapping for specific applications that manage their own window sizing
